## 🔍 프로젝트 정보
* React로 만든 첫번째 프로젝트로 React의 기본 훅들과 구조설계를 학습하기 위한 목적으로 시작했다.  
* 개발기간: 2023.12.13 ~ 2024.1.13
<br/>

## 📖 주요기능
* 랜덤하게 블록을 생성하는 기능 구현  
* 블록 회전함수 구현  
* Redux를 이용하여 Stage에 따른 Time값과 Grid Map 데이터 관리  
<br/>

## 🔥 개발 중 겪었던 이슈
* 구조설계에 대한 고민
* 이벤트핸들러 사용 시 closure로 인한 이슈
* 전역데이터의 상태관리
